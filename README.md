## Python File Manager (CRUD)
A simple, user-friendly command-line tool built with Python to manage local files. This project demonstrates basic File I/O, Pathlib for directory handling, and Exception Handling to prevent crashes.
## Features
# Create: Generate new files and save initial content.
# Read: List all files in the directory and view their contents.
# Update:
- Replace specific words/content within a file.
- Overwrite a file with entirely new data.
- Append new information to the end of a file.
# Delete: Remove unwanted files from the system.

Tech Stack
Language: Python 3.x
Modules: os, pathlib

## Example Usage
# When you run the script, follow the on-screen menu:
Enter 1 to create a new .txt file.
Enter 2 to see a list of files and pick one to read.
Enter 3 to update/modify an existing file.
Enter 4 to delete a file permanently
